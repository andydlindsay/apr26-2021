### Jest vs Cypress
* Jest
  * command line test runner
  * unit / integration testing
  * writing expectations
  * blazingly fast
* Cypress
  * runs a browser for each test
  * visit your website like a user
  * lots of overhead and takes a while
  * E2E testing === user stories
